# market-place

Overview

Users can browse, list, and manage products.
Users can place orders for available products.
Users can track their placed and received orders.


Tech Stack

Frontend: React.js
Backend: Node.js, Express.js
Database: Airtable
API Calls: Axios / Fetch API
